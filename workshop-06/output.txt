Процедура Val(St,IBR,Code) преобразует значение
St в величину целочисленного
или вещественного типа и помещает
значение в переменную IBR. Значение St
не должно содержать незначащих пробелов
в начале и в конце. Code
целочисленная переменная. Если во
время операции преобразования ошибки
не обнаружено, значение Code равно нулю,
если ошибка обнаружена, Code будет
содержать номер позиции
первого ошибочного символа, а значение
IBR не определено.
